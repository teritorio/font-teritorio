# Font Teritorio Map

A map icon font set by [Teritorio](teritorio.fr).

## Build

```
bash ./scripts/update.sh
```

## Publish

```
$ git checkout master
$ npm version minor -m "release: %s"
$ git push --follow-tags
$ npm publish
```
